# Cavalier Quick Draw


<img src="static/icons/android-chrome-512x512.png" alt="Cavalier Quick Draw Logo" width="200"/>

Simple game where you have to shoot the other player before they shoot you!

## How to play
1. Each player needs a phone
2. Go to website and host a game on a laptop (Make sure to turn up the volume)
3. Each player joins the game using their phone
4. When the game starts, each player has to wait for the "Fire!" signal
